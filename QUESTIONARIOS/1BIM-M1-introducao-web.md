# Exercícios

## 1 — Pergunta conceitual

Explique com suas próprias palavras:

* O que é **HTTP**

HTTP é um tipo de “regra” usada na internet para o navegador conversar com o servidor de um site. É por meio dele que você consegue acessar páginas, ver imagens e usar sites.

* Qual é a diferença entre **HTTP e HTTPS**

A diferença é que o HTTPS é mais seguro.  
O HTTP envia as informações de forma aberta, enquanto o HTTPS protege esses dados usando criptografia.

Explique também por que o **HTTPS é importante para a segurança na Web**.

O HTTPS é importante porque protege as informações do usuário, como senhas e dados pessoais. Assim, fica mais difícil alguém roubar esses dados durante o acesso a um site.

## 2 — Pergunta de aplicação

Imagine que uma empresa precisa criar um **site institucional**.

Explique:

* qual é o papel do **navegador**

O navegador é o programa que você usa para acessar sites, como Chrome ou Firefox. Ele pede as informações do site e mostra tudo na tela.

* qual é o papel do **servidor web**

O servidor é onde o site fica guardado. Ele recebe o pedido do navegador e envia os arquivos do site.

* como ocorre a comunicação entre eles quando um usuário acessa o site.

Quando você digita o endereço de um site, o navegador faz um pedido ao servidor. O servidor responde enviando os arquivos da página, e o navegador monta o site na tela.

Utilize exemplos para ilustrar sua explicação.

Por exemplo, quando você entra em um site, o navegador pede a página ao servidor. O servidor envia o conteúdo, e o navegador mostra o site para você.
